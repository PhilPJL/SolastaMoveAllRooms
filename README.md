# Solasta Move All Rooms

A simple mod to allow you to move all rooms in Solasta Dungeon Maker up/down/left/right.
When editing your dungeon the mod uses the keys bound to Select Character 1,2,3,4
Select character 1 = F1 or 1 = Right
Select character 2 = F2 or 2 = Left
Select character 3 = F3 or 3 = Up
Select character 4 = F4 or 4 = Down

Available at: https://www.nexusmods.com/solastacrownofthemagister/mods/tbd
