# Move All Rooms

A simple mod to allow you to move all rooms in Solasta Dungeon Maker up/down/left/right and rotate clockwise/counter-clockwise.

When editing your dungeon the mod uses the keys bound to Select Character 1, 2, 3, 4 and rotate CW, CCW.
* Use 'F1' or '1' to move rooms right by 1 cell, or 5 cells when pressing shift
* Use 'F2' or '2' to move rooms left by 1 cell, or 5 cells when pressing shift
* Use 'F3' or '3' to move rooms up by 1 cell, or 5 cells when pressing shift
* Use 'F4' or '4' to move rooms down by 1 cell, or 5 cells when pressing shift
* Use 'Q' to rotate clockwise
* Use 'E' to rotate counter clockwise

Available at: https://www.nexusmods.com/solastacrownofthemagister/mods/24

Before opening the solution
* Create the environment variable SolastaInstallDir = 'path to solasta', e.g. SolastaInstallDir = 'D:\Program Files (x86)\SteamLibrary\steamapps\common\Slasta_COTM'
* Ensure UnityModManager 0.23 or later is configured for Solasta
